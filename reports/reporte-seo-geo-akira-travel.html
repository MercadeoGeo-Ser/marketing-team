Diagnóstico Inicial

01

Estado actual de akiratravel.com

🚨 Problema crítico detectado: El servidor devuelve error HTTP 403 (Acceso Denegado) a rastreadores externos. Si Googlebot recibe este mismo error, la indexación del sitio está severamente comprometida.

Adicionalmente, Google tiene indexadas 3 versiones distintas del mismo dominio, lo que divide la autoridad acumulada:

  • http://www.akiratravel.com/
  • https://www.akiratravel.com/
  • https://akiratravel.com/
⚠️ Impacto: Cada URL duplicada recibe solo una fracción de la autoridad total. En lugar de tener 100% de fuerza en una URL, el dominio está repartido en 3, reduciendo el poder de posicionamiento.

SEO Técnico — Fundamentos

04

Conectar Google Search Console

¿Qué es? La herramienta gratuita de Google que muestra exactamente cómo ve Google tu sitio: qué páginas indexa, qué errores encuentra y qué keywords generan clics.

URL: search.google.com/search-console

Pasos:

  1. Ir a Search Console → "Añadir propiedad" → ingresar akiratravel.com
  2. Elegir verificación por DNS (la más confiable) o archivo HTML
  3. Esperar 24-48 horas para datos completos
  4. Revisar inmediatamente: Cobertura → páginas con error 404, páginas excluidas, páginas indexadas
  5. Revisar: Core Web Vitals → estado móvil y desktop
  6. Ir a Sitemaps → añadir akiratravel.com/sitemap.xml
05

Corregir versión canónica del dominio

Versión correcta objetivo: https://akiratravel.com (HTTPS, sin www)

Agregar en el archivo .htaccess del servidor (o configurar en cPanel/hosting):

RewriteEngine On

# Redirigir HTTP → HTTPS
RewriteCond %{HTTPS} off
RewriteRule ^(.*)$ https://%{HTTP_HOST}%{REQUEST_URI} [L,R=301]

# Eliminar www
RewriteCond %{HTTP_HOST} ^www\.akiratravel\.com [NC]
RewriteRule ^(.*)$ https://akiratravel.com/$1 [L,R=301]
✓ Resultado: Toda la autoridad acumulada en URLs antiguas se concentra en la versión correcta.
06

Verificar y corregir robots.txt

El robots.txt controla qué pueden rastrear los bots. Verificar en: akiratravel.com/robots.txt

Contenido correcto del archivo:

Allow: /

# Bots de búsqueda tradicional

Allow: /

🚨 Nunca debe aparecer: Disallow: / de forma general ni bloqueando carpetas de tours o productos.
07

Crear o corregir Sitemap XML

El sitemap le dice a Google exactamente qué páginas existen. En WordPress: instalar Yoast SEO o RankMath (gratuitos) — el sitemap se genera automáticamente.

Luego enviar en Google Search Console → Sitemaps → añadir sitemap.xml

Meta: El sitemap debe incluir todas las páginas de destinos, tours y artículos del blog. Verificar que no incluya páginas con noindex.

16

Implementar Schema Markup (datos estructurados)

El schema markup aumenta la visibilidad en IA entre un 30-40%. Se implementa como código JSON-LD en el <head> de cada página.

Schema TravelAgency — Página principal:

{
  "@context": "https://schema.org",
  "@type": "TravelAgency",
  "name": "Akira Travel",
  "description": "Agencia mayorista de viajes con sede en Medellín. Especialistas en paquetes a Europa, Asia y Medio Oriente desde Colombia.",
  "url": "https://akiratravel.com",
  "telephone": "+57-XXXXXXXXX",
  "address": {
    "@type": "PostalAddress",
    "addressLocality": "Medellín",
    "addressRegion": "Antioquia",
    "addressCountry": "CO"
  },
  "identifier": { "@type": "PropertyValue", "name": "RNT", "value": "24740" },
  "areaServed": "Colombia"
}

Schema FAQPage — Cada landing de destino:

{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [{
    "@type": "Question",
    "name": "¿Cuánto cuesta un paquete a Europa desde Colombia?",
    "acceptedAnswer": {
      "@type": "Answer",
      "text": "Los paquetes a Europa desde Colombia en Akira Travel parten desde USD $1.671
               por persona en acomodación doble, incluyendo vuelo, hotel 3-4 estrellas
               y guía en español."
    }
  }]
}
Cómo implementar en WordPress: Instalar plugin RankMath (gratuito) → Schema → añadir tipo TravelAgency. Verificar en: search.google.com/test/rich-results

SEO Local + Reseñas

19

Optimizar Google Business Profile

El perfil de Google Business es lo que aparece en Google Maps y en el panel lateral cuando alguien busca "Akira Travel" o "agencia de viajes Medellín".

  1. Ir a business.google.com → reclamar o crear el perfil
  2. Categoría principal: Agencia de viajes · Secundaria: Operador turístico
  3. Descripción (750 caracteres): incluir "mayorista", "RNT 24740", "Medellín", destinos principales
  4. Subir mínimo 20 fotos: oficina, equipo, destinos, grupos de viajeros
  5. Verificar que nombre, dirección y teléfono sean idénticos en el sitio web y en Google Business
  6. Publicar actualizaciones semanales en Google Business (mismas que Instagram)
  7. Responder todas las reseñas en menos de 48 horas
